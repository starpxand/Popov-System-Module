# Popov-System-Module
# Модуль навигации по семантическому графу

### Описание модуля
Модуль предназначен для алгоритмической реализации и визуализации механизмов навигации в сложных структурах данных. Основная задача — поиск кратчайших путей между объектами (вершинами) и обнаружение логических ловушек (циклов).

Ключевые функции:
* Нахождение кратчайшего пути от узла "Студент" до критических узлов системы.
* Масштабируемый поиск смежных узлов на заданную глубину.
* Генерация кода визуализации в формате DOT для Graphviz.
* Автоматическая детекция циклических ссылок.

### Стек технологий
* Язык реализации: C++
* Язык описания сценариев: Gherkin (BDD)
* Инструменты визуализации: Graphviz (формат DOT)
* Среда разработки: Visual Studio Code
*  Прототипирование UI: Pixso / Figma

### Интерфейс (UI); (Документация и макеты)
Макеты интерфейса, разработанные в Pixso, доступны в папке /docs.
