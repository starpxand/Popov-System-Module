# Popov-System-Module
# Модуль навигации по семантическому графу

### Описание модуля
[span_4](start_span)Модуль предназначен для алгоритмической реализации и визуализации механизмов навигации в сложных структурах данных[span_4](end_span). [span_5](start_span)[span_6](start_span)Основная задача — поиск кратчайших путей между объектами (вершинами) и обнаружение логических ловушек (циклов)[span_5](end_span)[span_6](end_span).

Ключевые функции:
* [span_7](start_span)Нахождение кратчайшего пути от узла "Студент" до критических узлов системы[span_7](end_span).
* [span_8](start_span)Масштабируемый поиск смежных узлов на заданную глубину[span_8](end_span).
* [span_9](start_span)Генерация кода визуализации в формате DOT для Graphviz[span_9](end_span).
* [span_10](start_span)[span_11](start_span)Автоматическая детекция циклических ссылок[span_10](end_span)[span_11](end_span).
